# QuadraticSolver

A simple web calculator for quadratic equations (ax² + bx + c = 0).  
Built with HTML, CSS, and JavaScript.

---

## Features

- Checks that all coefficients are filled
- Validates number inputs
- Ensures a ≠ 0
- Calculates the discriminant
- Shows real roots (if any), rounded to 2 decimal places

---

## Usage

1. Open the page in your browser  
2. Enter coefficients `a`, `b`, and `c`  
3. Click **Solve**  
4. View the result

---

## Equation

ax² + bx + c = 0
x = (-b ± √(b² − 4ac)) / (2a)
